If Expression

If Expression

  • expression returns a value
  • no need for ternary operations
val taskDone = true
val remarks = if (taskDone) "Great!" else "Keep it up!"
println(remarks)

More information about control flow here: https://kotlinlang.org/docs/reference/control-flow.html
