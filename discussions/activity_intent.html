Start an Activity with an Intent

  1. Open activity_main.xml and add an id to the TextView. Example: text_hello_world

    xmlns:tools="http://schemas.android.com/tools"

        android:layout_height="wrap_content"
        android:text="Hello World!"
        app:layout_constraintBottom_toBottomOf="parent"

        app:layout_constraintRight_toRightOf="parent"
        app:layout_constraintTop_toTopOf="parent" />

  1. Open activity_second.xml and add a TextView. You can copy and paste the TextView from activity_main.xml

  1. Assign an id to the TextView. Example: text_hello_second_activity
  2. One way to trigger a redirect to a new activity/ screen is through a click event. Add an onClickListener in MainActivity
package com.allsortsofrandom.todaysactivity

import android.content.Intent
import android.support.v7.app.AppCompatActivity
import android.os.Bundle
import kotlinx.android.synthetic.main.activity_main.*

class MainActivity : AppCompatActivity() {

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)

        text_hello_world.setOnClickListener{
            val secondActivityIntent = Intent(this, SecondActivity::class.java)
            startActivity(secondActivityIntent)
        }
    }
}
  1. Run the app
